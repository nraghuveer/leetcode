Gangs of Primes and Composites
Given a two dimensional integer matrix, find the number of prime and composite gangs. A collection of integers of the same type viz. prime or composite forms a gang where adjacency is assumed vertical and horizontal.

Input
The first line of input consists of an integer t which is the number of test cases. First line of each test case consists of m rows and n columns in the test case space separated. Next m rows follow, each consisting of n space separated integers.

Output
For each test case, print the number of prime and composite gangs space separated.

Constraints
1 <= t <= 1000

1 <= m,n <= 100

2 <= integer <= 10000
